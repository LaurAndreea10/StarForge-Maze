# ✦ StarForge Maze

**StarForge Maze** is a bilingual arcade maze game in which you guide an energy core, collect every star and key, avoid traps, and unlock the exit portal.

**StarForge Maze** este un joc arcade bilingv în care conduci un nucleu energetic prin labirint, colectezi toate stelele și cheia, eviți capcanele și deblochezi portalul de ieșire.

## 🌐 Live demo

[Play StarForge Maze](https://laurandreea10.github.io/StarForge-Maze/)

## ✨ Features / Funcționalități

- Romanian and English interface with a persistent language selector
- Interfață în română și engleză, cu salvarea limbii selectate
- Procedurally generated maze gameplay
- Multiple game modes: Campaign, Challenge, and Survival
- Three difficulty levels: Easy, Normal, and Hard
- Stars, keys, crystals, portals, traps, bonuses, and energy management
- Scanner, shield, magnet, teleport, undo, and hint tools
- Score, combo, lives, timer, level, mission progress, and local best score
- Keyboard and touch controls
- Pause, restart, help dialog, sound, and theme controls
- Responsive layout and accessibility-friendly elements
- Local persistence through `localStorage`

## 🎮 How to play / Cum se joacă

1. Move the energy core through the maze.
2. Collect all stars and recover the key.
3. Avoid red traps or activate the shield.
4. Use tools and bonuses strategically.
5. Enter the portal after completing the mission objectives.

---

1. Deplasează nucleul energetic prin labirint.
2. Colectează toate stelele și recuperează cheia.
3. Evită capcanele roșii sau activează scutul.
4. Folosește strategic uneltele și bonusurile.
5. Intră în portal după îndeplinirea obiectivelor.

## ⌨️ Controls / Comenzi

| Action / Acțiune | Control |
|---|---|
| Move / Deplasare | `WASD` or / sau `↑ ↓ ← →` |
| Pause / Pauză | `P` |
| Restart | `R` |
| Tools / Unelte | `1–4` |
| Undo / Înapoi | `U` |
| Hint / Indiciu | `H` |

Touch controls are available on mobile devices.

Comenzile tactile sunt disponibile pe dispozitivele mobile.

## 🧰 Tools / Unelte

- **Scanner** — reveals useful information about the maze
- **Shield / Scut** — protects the player from a trap
- **Magnet** — helps collect nearby objects
- **Teleport** — moves the player to another position
- **Undo / Înapoi** — reverses the previous move
- **Hint / Indiciu** — suggests a useful direction
